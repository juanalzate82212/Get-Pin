# Get-Pin
This project generates all possible combinations of a PIN entered on a numeric keypad, considering that each digit can be the one pressed or an adjacent one (horizontally or vertically). The system returns all valid variations sorted in ascending order, simulating the behavior of a digital lock.

Proyecto que genera todas las combinaciones posibles de un PIN observado en un teclado numérico, considerando que cada dígito puede ser el presionado o uno adyacente (horizontal o vertical). El sistema devuelve todas las variaciones válidas ordenadas de forma ascendente, simulando el comportamiento de una cerradura digital.
